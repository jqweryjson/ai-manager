https://avatarger.ru/
Разрабатываю, в свободное время, свой сервис для автоматизации работы с клиентами в мессенджерах, соц-сетях и CRM системах. На данный момент упор сделан на интеграцию с Telegram как user-bot, которая позволяет гибко настроить стратегию автоответов в выбранных чатах на основе любых данных загружаемых в векторную бд. Финальный ответ готовит внешний LLM провайдер. Учтена обработка flood-wait и большого колличества событий, используя очередь Rabbit-MQ.

Веду разработку системы полностью один от дизайна до devops, монетизации и продвижения. Инвестиции на данный момент - это только оплата VPN, VPS и хостинга

Стек проекта:

Frontend: SPA React-19\TS\Vite
Backend: Nodejs (Fastify)\Zod\Postgres\Milvus(VectorDB)\Python(Bge-m3 - vector)\redis\Rabbit-MQ\LLM integration (Grok-api)\ShadowSocks-proxy\Docker\CI-CD
