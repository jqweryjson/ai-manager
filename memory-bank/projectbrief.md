Цель MVP
Ответы на вопросы сотрудников на основе внутренних текстов компании, с обязательными цитатами и без «додумываний».

Ключевые задачи

- Приём текстов (txt/markdown), автоматическая обработка и индексирование.
- Поиск релевантных фрагментов и сборка ответа строго на их основе.
- Простое API для чата и загрузки данных.

Стек

- Fastify.js (API), Milvus (векторы), Redis (кэш/сессии), Groq API (LLM), Docker.

Ключевые параметры (на старте)

- chunk ≈800, overlap ≈18%, top_k=30, coverage_threshold=0.6.

Критерии успеха

- ≥80% ответов с валидными цитатами на эталонном наборе из 20 вопросов.
- Среднее время ответа < 2 с при локальной нагрузке.

Границы

- Только текстовые данные и один профиль эмбеддингов; PDF/CRM позже.
- Персональная база знаний: данные строго личные, общего доступа/шаринга нет.
