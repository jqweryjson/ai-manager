# Настройка Telegram Login Widget для веб-версии

## Что реализовано

1. ✅ Backend эндпоинт `/api/auth/telegram-web` для обработки авторизации
2. ✅ Валидация подписи от Telegram Login Widget
3. ✅ Frontend компонент `TelegramAuthButton` с официальным виджетом
4. ✅ Интеграция в страницу авторизации

## Настройка

### 1. Получить bot_username

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Если бот еще не создан, отправьте `/newbot` и следуйте инструкциям
3. Сохраните **имя бота** (username, например `my_bot`, без @)

### 2. Привязать домен к боту

1. В чате с @BotFather отправьте команду `/setdomain`
2. Выберите вашего бота
3. Укажите домен вашего сайта (например: `ai-manager-front.onrender.com` или `localhost:8002` для разработки)

**Важно:** Домен должен быть привязан, иначе виджет не будет работать.

### 3. Настроить переменные окружения

**Backend (.env):**

```env
TELEGRAM_BOT_TOKEN=your_bot_token_from_botfather
```

**Frontend (.env или .env.local):**

```env
VITE_TELEGRAM_BOT_USERNAME=your_bot_username
```

Где `your_bot_username` - это имя бота без @ (например: `my_bot`)

### 4. Проверка работы

1. Запустите backend и frontend
2. Откройте страницу `/auth`
3. Должна появиться кнопка "Войти через Telegram" (официальный виджет)
4. При клике откроется окно авторизации Telegram
5. После авторизации пользователь будет перенаправлен в приложение

## Как это работает

1. **Пользователь нажимает кнопку** → открывается официальный виджет Telegram
2. **Пользователь авторизуется** → Telegram передает данные в глобальную функцию `onTelegramAuth(user)`
3. **Frontend отправляет данные** → POST `/api/auth/telegram-web` с данными пользователя
4. **Backend валидирует подпись** → проверяет HMAC-SHA256 подпись
5. **Backend создает/находит пользователя** → по `telegram_id`
6. **Backend возвращает токены** → JWT access/refresh токены
7. **Frontend сохраняет токены** → и перенаправляет в приложение

## Преимущества

- ✅ Один `telegram_id` для веб и мобильного → один аккаунт
- ✅ Бесшовная авторизация между устройствами
- ✅ Не нужен SMS-провайдер
- ✅ Официальный виджет Telegram (безопасно)

## Отладка

Если виджет не появляется:

1. Проверьте, что `VITE_TELEGRAM_BOT_USERNAME` установлен
2. Проверьте, что домен привязан к боту через `/setdomain`
3. Проверьте консоль браузера на ошибки

Если авторизация не работает:

1. Проверьте, что `TELEGRAM_BOT_TOKEN` установлен на backend
2. Проверьте логи backend на ошибки валидации
3. Убедитесь, что домен совпадает с привязанным в BotFather
